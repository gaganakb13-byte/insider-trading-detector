# Insider Trading Detector 🔍📈

AI-powered web app to detect suspicious trading activity using:
- Kaggle datasets (historical anomalies)
- Yahoo Finance (real-time anomalies)
- Sentiment analysis (TextBlob)
- Random Forest ML model
- Email alerts via Gmail
